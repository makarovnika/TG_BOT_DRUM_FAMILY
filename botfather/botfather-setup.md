# Настройка профиля бота в @BotFather

Пошаговая инструкция применения дизайн-ТЗ (`drum-family-bot-tz.md` §7) к
профилю бота **@DrumFamily_Tomsk_Bot**. Все шаги выполняются в чате с
[@BotFather](https://t.me/BotFather).

## 0. Подготовка

- В этом чате запусти @BotFather и убедись, что бот `@DrumFamily_Tomsk_Bot`
  виден в списке (`/mybots` → выбрать его).
- Аватар и баннеры пока в работе — установка аватара ниже отложена, пока
  не будет финального ассета (см. §5 ТЗ).

## 1. Имя и username

✅ Уже верно. Имя: `Drum Family Томск`. Username: `DrumFamily_Tomsk_Bot`.
Менять не надо.

## 2. Аватар (`/setuserpic`)

Когда будет готов файл `assets/avatar/bot-avatar-512.png` (см. ТЗ §5):

1. `/setuserpic` в @BotFather
2. Выбрать бота `@DrumFamily_Tomsk_Bot`
3. Прислать файл `bot-avatar-512.png` как **фото** (не как «файл», иначе
   Telegram не примет).

## 3. Bio (`/setdescription`)

Видно в карточке бота, до 120 символов.

1. `/setdescription` → выбрать бота.
2. Прислать текстом:

```
🥁 Запись на пробный и регулярные занятия. Drum Family, Томск. пр. Комсомольский 68/5
```

## 4. About (`/setabouttext`)

Видно до старта бота. До 512 символов.

1. `/setabouttext` → выбрать бота.
2. Прислать:

```
Привет! Это бот барабанной школы Drum Family в Томске.

Здесь можно записаться на пробный урок, выбрать удобное время, посмотреть расписание и связаться с администратором.

Сайт: drumfamily.ru
Адрес: пр. Комсомольский, 68/5
Тел.: +7 (995) 292-81-03
```

## 5. Команды (`/setcommands`)

Заметка: команды программно регистрируются на каждом старте бота
(`Bot.set_my_commands` в `src/main.py`). Этот шаг — для дублирующей
конфигурации на стороне BotFather (так Telegram-клиенты могут увидеть
список даже когда бот не запущен).

1. `/setcommands` → выбрать бота.
2. Прислать ровно этот блок:

```
start - 🥁 Начать
trial - Записаться на пробный
schedule - Моё расписание
prices - Стоимость занятий
contacts - Адрес и контакты
faq - Частые вопросы
admin - Написать администратору
profile - Мой профиль
cancel - Отменить текущее действие
help - Подсказка по командам
```

## 6. Меню-кнопка (`/setmenubutton`)

Mini App в проекте пока нет. Оставить **дефолт** («команды») — она будет
показывать список из шага 5.

Если в будущем добавим Mini App:
1. `/setmenubutton` → выбрать бота → выбрать кнопку с URL.
2. Текст кнопки: `🥁 Записаться`.
3. URL: ссылка на Mini App (например, `https://drumfamily.ru/app/`).

## 7. Чек после настройки

- Открыть `@DrumFamily_Tomsk_Bot` в Telegram, перейти в карточку
  (кнопка-меню «···» → «View profile»).
- Должно отображаться:
  - имя «Drum Family Томск»;
  - аватар (когда установим);
  - bio из шага 3;
  - about — видно до старта или по `View bio`.
- При вводе `/` в чате должен появиться выпадающий список из 10 команд
  (шаг 5).
- Все команды должны открываться без ошибок.

`drum-family-bot-mockup.html` (header чата): в шапке должна быть круглая
